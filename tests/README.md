# Tests

This folder is not active behavior.

Contents here are audit reports, smoke test evidence, regression test results, and verification material.

Use this folder for testing and audit evidence only. Do not treat audit reports or test outputs as active rules.

Default active source of truth is:

`current/`

Default package checks:

- `python3 package_linter.py`
- `python3 scripts/validate_package_guard.py`

Use `tests/smoke_tests/package_integrity_checklist.md` as the non-active checklist for package audit evidence.
